# Support

This is a portfolio/demo repository.

## Getting Help

- Open a GitHub issue with:
  - what you were trying to do
  - expected vs actual result
  - logs/screenshots (do not include secrets)
  - environment details (OS, Python version, `az`/`gh` versions)

## Azure Cost Safety

If you deployed to Azure, use:

- `scripts/azure/inventory.sh` to see what's running
- `scripts/azure/teardown.sh` to delete the resource group

# Security Policy

## Reporting a Vulnerability

This repository is a portfolio/demo project.

If you believe you found a security vulnerability:

1. Do **not** open a public issue with exploit details.
2. Contact the repository owner via GitHub (preferred) and include:
   - a short description of the issue
   - steps to reproduce
   - affected endpoints/files
   - impact assessment (what an attacker can do)

## Supported Versions

Only the latest `main` branch is supported.

## Secrets

- Do not commit secrets to this repository.
- Rotate keys/tokens immediately if they were pasted in chat or committed accidentally.
